INSTRUCCIONES PARA EJECUTAR EL PROGRAMA:

1º. HACER "cmake.." DESDE EL TERMINAL, EN LA CARPETA "build".
	-Dentro de esa carpeta se generarán los archivos necesarios del CMake, así como el Makefile.

2º. HACER "make" DESDE EL TERMINAL, EN LA CARPETA "build" (ES DÓNDE SE ENCUENTRA EL MAKEFILE).

3º. A CONTINUACIÓN UNA VEZ COMPILADO EL PROGRAMA, DAMOS PERMISO DE EJECUCIÓN: "chmod u+x grafica~~~.sh", A CADA UNO DE LOS SCRIPTS CREADOS PARA GENERAR LAS GRÁFICAS (ESTOS SE ENCUENTRAN EN LA CARPETA PRINCIPAL).

4º. POR ÚLTIMO, UNA VEZ MÁS DESDE EL TERMINAL, EN LA CARPETA "build", ejecutamos el archivo ejecutable que hemos generado al hacer los paso 1 y 2.
